HTML теги с нуля

Основные (простые) теги, которые вам нужно знать:

Минутку внимания, если у вас на устройстве включен "Режим затемнения" или "Ночной режим", то для более лучшей читаемости текста рекомендуем отключить его, потому что изза этого режима цвета сайта отображаются неправильно! Приятного чтения!

1. <a> - тег для встраивания ссылок, закрывается вторым тегом </a>, пример использования:

<a href="ссылка на сайт"> Текст в который будет встроенна ссылка </a>

2. <p> - тег который создает текст с абзаца, закрывается вторым тегом </p>, пример использования:

<p> текст </p>

3. <h1> - тег который создает заголовок 1 уровня (самый большой), всего таких тегов 6 <h1>, <h2>, <h3>, <h4>, <h5>, <h6> (чем выше значение - тем меньшего размера будет текст заголовка), закрывается вторым тегом </h1>, пример использования:

<h1> текст </h1>

4. <i> - тег который делает ваш текст курсивным, работает только с тегами, которые создают текст (<p> например), закрывается вторым тегом, пример использования:

<p><i> текст </i></p>

5. <b> - тег который делает ваш текст жирным, работает только с тегами, которые создают текст (<p> например; тег <b> ничем не отличается от того же <i>), закрывается вторым тегом, пример использования:

<p><b> текст </b></p>

6. <u> - тег который делает ваш текст подчеркнутым, работает только с тегами, которые создают текст (<p> например; тег <u> ничем не отличается от вышеперчисленых: <b>, <i>), закрывается вторым тегом, пример использования:

<p><u> текст </u></p>

7. <font> - тег который добавляет текст на сайт (без абзаца), закрывается вторым тегом, пример использования:

<font> текст <font>

8. <li> - тег <li> определяет отдельный элемент списка. Внешний тег <ul> или <ol> устанавливает тип списка — маркированный или нумерованный.

<ul><li> элемент маркированного списка </li></ul>
Или
<ol><li> элемент нумерованного списка </li></ol>

9. <ruby> - Тег <ruby> предназначен для добавления небольшой аннотации сверху или снизу от заданного текста. Такая форма записи преимущественно используется для идеографической письменности вроде китайского языка, но может применяться и для других языков, если требуется написать один текст над другим.

<ruby> текст <rt>аннотация</rt></ruby>

Теги, которые не закрываются вторым тегом:

1. <hr> - тег который создает горизонтальную линию, НЕ закрывается вторым тегом, пример использования:

<hr>

2. <img> - тег который встраивает картинку на вашу страницу, НЕ закрывается вторым тегом, пример использования:

<img src="ссылка на картинку с компьютера (место где она лежит на диске)">

3. <br> или </br> - тег который создаёт абзац (без текста, просто пропуск), НЕ закрывается вторым тегом, пример использования:

<br> или </br>

4. <!DOCTYPE> - Элемент <!DOCTYPE> предназначен для указания типа текущего документа — DTD (document type definition, описание типа документа). Это необходимо, чтобы браузер понимал, как следует интерпретировать текущую веб-страницу, поскольку HTML существует в нескольких версиях, кроме того, имеется XHTML (EXtensible HyperText Markup Language, расширенный язык разметки гипертекста), похожий на HTML, но различающийся с ним по синтаксису. Чтобы браузер «не путался» и понимал, согласно какому стандарту отображать веб-страницу и необходимо в первой строке кода задавать <!DOCTYPE>.

<!DOCTYPE html>

5. <input> - Тег <input> является одним из разносторонних элементов формы и позволяет создавать разные элементы интерфейса и обеспечить взаимодействие с пользователем. Главным образом <input> предназначен для создания текстовых полей, различных кнопок, переключателей и флажков. Хотя элемент <input> не требуется помещать внутрь контейнера <form>, определяющего форму, но если введенные пользователем данные должны быть отправлены на сервер, где их обрабатывает серверная программа, то указывать <form> обязательно. То же самое обстоит и в случае обработки данных с помощью клиентских приложений, например, скриптов на языке JavaScript.

<input type="radio" name="browser" value="ie">

Атрибуты, что это такое, где применять, как выглядят и т.д:

Начнём с того, что Атрибуты это не отдельные теги и их нельзя писать просто самих по себе, они пишутся "внутрь" тега, также атрибуты не закрываются и пишутся в "открывающемся теге" (т.е внутрь тега <p>, а не </p> например) сейчас вы сможете увидеть базовые атрибуты для html тегов

1. href - базовый атрибут для тега <a> в который нужно написать ссылку на сайт (если вы забыли про тег <a> перечитайте еще раз у нас на сайте) пример:

<a href="ссылка на сайт"> Текст в который будет встроенна ссылка </a>

Как мы уже ранее писали Атрибут href встроен в первый или "открывающий" тег <a>

2. src - базовый атрибут для тега <img> (этот тег позволяет добавить картинку на сайт) в который нужно написать ссылку на картинку (эта картинка должна быть у вас скачана, после чего вы должны указать на неё путь через ваш диск, почему она должна быть скачана? - Атрибут src означает относительный или абсолютный путь к файлу на вашем компьютере, также должен быть указан формат или расширение файла например .jpg .png .jpeg и т.д.) пример:

<img src="напишите ссылку на картинку с пк, images/aboba.jpg например"></img>

3. color - базовый атрибут для многих тегов который меняет цвет, но в основном используется для тега <font> (этот тег добавляет текст) пример:

<font color="white, blue, red, green и т.д"> Сам текст, который будет добавлен на сайт </font>

4. align - "уникальный" атрибут для многих тегов который меняет расположение, но вам пока он нужен только для тега <p> (этот тег добавляет текст) пример:

<p align="left, middle, right и т.д"> Сам текст, который будет добавлен на сайт </p>
