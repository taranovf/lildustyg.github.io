HTML теги с нуля

Основные (простые) теги, которые вам нужно знать:

Минутку внимания, если у вас на устройстве включен "Режим затемнения" или "Ночной режим", то для более лучшей читаемости текста рекомендуем отключить его, потому что изза этого режима цвета сайта отображаются неправильно! Приятного чтения!

1. <a> - тег для встраивания ссылок, закрывается вторым тегом </a>, пример использования:

<a href="ссылка на сайт"> Текст в который будет встроенна ссылка </a>

2. <p> - тег который создает текст с абзаца, закрывается вторым тегом </p>, пример использования:

<p> текст </p>

3. <h1> - тег который создает заголовок 1 уровня (самый большой), всего таких тегов 6 <h1>, <h2>, <h3>, <h4>, <h5>, <h6> (чем выше значение - тем меньшего размера будет текст заголовка), закрывается вторым тегом </h1>, пример использования:

<h1> текст </h1>

4. <i> - тег который делает ваш текст курсивным, работает только с тегами, которые создают текст (<p> например), закрывается вторым тегом, пример использования:

<p><i> текст </i></p>

5. <b> - тег который делает ваш текст жирным, работает только с тегами, которые создают текст (<p> например; тег <b> ничем не отличается от того же <i>), закрывается вторым тегом, пример использования:

<p><b> текст </b></p>

6. <u> - тег который делает ваш текст подчеркнутым, работает только с тегами, которые создают текст (<p> например; тег <u> ничем не отличается от вышеперчисленых: <b>, <i>), закрывается вторым тегом, пример использования:

<p><u> текст </u></p>

Теги, которые не закрываются вторым тегом:

1. <hr> - тег который создает горизонтальную линию, НЕ закрывается вторым тегом, пример использования:

<hr>

2. <img> - тег который встраивает картинку на вашу страницу, НЕ закрывается вторым тегом, пример использования:

<img src="ссылка на картинку с компьютера (место где она лежит на диске)">

3. <br> или </br> - тег который создаёт абзац (без текста, просто пропуск), НЕ закрывается вторым тегом, пример использования:

<br> или </br>

Атрибуты, что это такое, где применять, как выглядят и т.д:

Начнём с того, что Атрибуты это не отдельные теги и их нельзя писать просто самих по себе, они пишутся "внутрь" тега, также атрибуты не закрываются и пишутся в "открывающемся теге" (т.е внутрь тега <p>, а не </p> например) сейчас вы сможете увидеть базовые атрибуты для html тегов

1. href - базовый атрибут для тега <a> в который нужно написать ссылку на сайт (если вы забыли про тег <a> перечитайте еще раз у нас на сайте) пример:

<a href="ссылка на сайт"> Текст в который будет встроенна ссылка </a>

Как мы уже ранее писали Атрибут href встроен в первый или "открывающий" тег <a>

2. src - базовый атрибут для тега <img> в который нужно написать ссылку на картинку (эта картинка должна быть у вас скачана, после чего вы должны указать на неё путь через ваш диск, почему она должна быть скачана? - Атрибут src означает относительный или абсолютный путь к файлу на вашем компьютере, также должен быть указан формат или расширение файла например .jpg .png .jpeg и т.д.) пример:

<img src="напишите ссылку на картинку с пк, images/aboba.jpg например"></img>

Как должен выглядеть код html страницы по стандарту:

<html>

<head>

<title>

название страницы

</title>

</head>

<body>

код, атрибуты, теги, и т.д.

</body>

</html>
